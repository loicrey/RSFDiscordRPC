# AGENTS.md

Read `README.md` first for repository context.

Read `CONTRIBUTING.md` before making changes.

Work from the repository root:

- Native plugin code lives under `src/`.
- Tracked plugin config lives under `src/config/`.
- Build and packaging scripts live under `src/build/`.
- Vendored native dependencies live under `src/third_party/`.
- The website and public RPC assets live under `web/`.
- Keep generated files, local build outputs, IDE state, and packaged releases out of the repository.
